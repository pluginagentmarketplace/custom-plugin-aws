# /assess - Knowledge Assessment & Evaluation

## Overview

The `/assess` command helps you evaluate your current knowledge level, identify skill gaps, and get personalized recommendations for improvement.

## Usage

```
/assess
/assess --skill api-design
/assess --agent frontend
/assess --roadmap react
/assess --all
```

## Assessment Features

### 1. Adaptive Difficulty

- **Easy Level** - Beginner concepts (20 questions)
- **Medium Level** - Intermediate knowledge (30 questions)
- **Hard Level** - Advanced topics (40 questions)

Questions automatically adjust based on your answers.

### 2. Question Types

- **Multiple choice** - 4 options
- **Multiple select** - Choose all correct answers
- **Code snippets** - Identify output or errors
- **Scenarios** - Real-world problem solving
- **Fill-in-the-blank** - Knowledge recall
- **Ordering** - Sequence correct steps

# /browse - Explore All 81 Developer Roadmaps

## Overview

The `/browse` command lets you explore and navigate all 81 available developer roadmaps organized by category and agent.

## Usage

```
/browse
/browse --category languages
/browse --agent frontend
/browse --search react
/browse --skill api-design
```

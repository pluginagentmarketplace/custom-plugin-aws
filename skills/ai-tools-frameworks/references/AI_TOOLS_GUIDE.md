# AI Tools & Frameworks Guide

## LLM Providers

| Provider | Models | Best For |
|----------|--------|----------|
| OpenAI | GPT-4, GPT-3.5 | General, coding |
| Anthropic | Claude 3 | Long context, safety |
| Google | Gemini | Multimodal |
| HuggingFace | Open models | Self-hosting |

## Frameworks

| Framework | Use Case |
|-----------|----------|
| LangChain | LLM orchestration |
| LlamaIndex | RAG pipelines |
| AutoGPT | Autonomous agents |
| CrewAI | Multi-agent systems |

## Prompt Engineering

1. **Be specific** - Clear instructions
2. **Provide context** - Background info
3. **Use examples** - Few-shot learning
4. **Chain of thought** - Step by step
5. **Format output** - JSON, markdown

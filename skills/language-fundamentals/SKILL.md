## Core Concepts Across Languages

### Data Types & Variables
- Primitive types (int, float, string, boolean)
- Collections (arrays, objects, maps, sets)
- Type systems (static vs dynamic, strong vs weak)
- Memory management (stack vs heap)

### Control Flow
- Conditionals (if/else, switch)
- Loops (for, while, forEach)
- Exception handling (try/catch/finally)
- Pattern matching (where available)

### Functions & Procedures
- Function declaration and invocation
- Parameters and return values
- Scope and closures
- Higher-order functions (callbacks, map, filter, reduce)
- Functional vs imperative styles

### Object-Oriented Programming
- Classes and objects
- Inheritance and composition
- Encapsulation and access modifiers
- Polymorphism and interfaces
- Method overloading/overriding

### Asynchronous Programming
- Callbacks
- Promises and async/await
- Event loops
- Concurrent execution patterns
